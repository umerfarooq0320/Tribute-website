# Tribute Website - Quaid-e-Azam

This is a simple tribute website dedicated to Quaid-e-Azam Muhammad Ali Jinnah.

## 📌 About Project

This project is built using HTML and CSS as part of an assignment. It highlights the life, achievements, and contributions of Quaid-e-Azam.

## 🛠️ Technologies Used

* HTML
* CSS

## 🎯 Purpose

To practice basic web development skills and create a tribute page.
